Kesimpulan atau hasil pertemuan 6 yang saya ambil adalah:

  1. Plugins merupakan suatu program mini yang dimasukan kedalam wordpress sehingga wordpress memiliki fitur-fitur tambahan contohnya seperti visitor counter.
  2. Dalam menginstal plugin dapat dilakukan melalui 3 cara yaitu dengan
  3. • Langsung online melalui wordpress atau https://wordpress.org/plugins/
  4. • Semi online,Dengan mengungah file zip plugin yang telah di download
  5. • Memindahkan folder plugins dari hasil download yang sudah di extract
  6. Cara menerapkan plugins contohnya visitor counter untuk wordpress dengan cara:
  7. • Ketikan” plugin visitor counter wordpress” di Google
  8. • Kemudian pilih plugins yang mendapat ulasan terbaik
  9. • Download plugins yang sudah dipilih,setelah berhasil di download
  10. • Extract file tersebut
  11. • Kemudian pindahkan folder yang sudah di extract ke dalam htdocs -> Folder Aplikasi->wp-contens->Plugin
  12. • Jika sudah akses pada halaman backend klik plugins
  13. • Cari Nama plugins yang dipilih tadi
  14. • Klik aktifkan
  15. • Pergi ke tampilan pilih widget
  16. • Kemudian drag nama plugin tadi ke dalam footer/siderbar ,maka plugin berhasil diterapkan
  17. Cara download plugins Wordfence untuk wordpres
  18. • Download file wordfence
  19. • Kemudian extract file hasil download Folder Aplikasi->wp-contens->Plugin
  20. • Masuk ke dalam akses halaman backend
  21. • Ke menu plugins,cari nama wordfence
  22. • Klik aktifkan,makan akan terdapat tambahan menu wordfence pada setting
